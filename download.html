Download

In this page you can find links to install the PhishDetect Browser Extension. It is available for Mozilla Firefox and Google Chrome (as well as Chromium). The browser extension is pre-configured to use a PhishDetect that we run. To configure a different node, please refer to Help pages.

Please Notice: PhishDetect is currently in beta! In order to use the default PhishDetect Node operated by the project's creators, you will be required to register to request a secret token.
