List of PhishDetect Nodes

This is a list of available PhishDetect Nodes. While PhishDetect's Browser Extension comes configured with a default Node, you might decide to switch to a different one that might better cater your needs.
Location Operator Authentication Description
phishdetect.securitywithoutborders.org Security Without Borders No This is the default public Node.
phishdetect.amnesty.tech Amnesty International Required This is a private node operated by Amnesty International's Security Lab, for exclusive use by Human Rights Defenders. Users are requested to register and be manually approved by the operators.
